# 서버
- 주소: `veneer.r-e.kr`
- 공용 계정: `kisi`
- 접속 방법: cmd 실행 -> `ssh kisi@veneer.r-e.kr` 접속 -> 비밀번호 입력 (yes/no/... 나오면 yes입력 후 비밀번호 입력)

 
# 환경
사용 도구
```
sudo apt install python3 python3-venv ffmpeg
```

파이썬 모듈
```
pip install flask flask-sslify pydub whisper-timestamped
```
